# Debug Response Encryption Public Keys

This directory stores public keys used by SDKs to encrypt debug response
loglines for later inspection and debugging by CU Devs Collective maintainers.

## Why?

You can find more information in the [Reporting Issues for Codegen Clients](https://github.com/cu-devs-collective/open-cu-services-sdk#reporting-issues-for-codegen-clients)
section.
